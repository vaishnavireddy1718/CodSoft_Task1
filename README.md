# 📱 Mobile App Signup Flow

## 📌 Overview

This project was created as part of the **CodSoft UI/UX Design Internship**.

The objective of this task was to design a modern, user-friendly, and intuitive mobile application signup flow that provides a smooth onboarding experience while making account creation simple, engaging, and efficient.

## ✨ Features

- Modern and clean user interface
- Simple onboarding experience
- Multi-step signup process
- User type selection
- Skill selection screen
- Career goal selection
- Success confirmation screen
- Minimal and user-friendly design
- Consistent color palette
- Easy navigation
